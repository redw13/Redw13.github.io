指令 说明
hexo new "文章标题" 创建文章标题，在 source/_posts 目录下生成 .md 文件
hexo n 文章标题 创建文章名
hexo generate 生成public 目录下所有的静态文件。
hexo g 生成静态文件
hexo server 本地启动服务器，默认端口: 4000。默认地址
hexo s 启动本地服务器（ctrl+c 关闭本地服务器）
hexo clean 清除 public 目录下的所有文件。
hexo c 清理缓存文件
hexo deploy 推送public 目录下的所有文件到指定服务器。
hexo d 上传服务器
hexo clean && hexo g && hexo s 清理缓存+生成静态文件+启动本地服务器
hexo clean && hexo g && hexo d 清理缓存+生成静态文件+上传指定服务器
